Mod Admin Aktif

Panduan Mengaktifkan Sistem Live Score (Awan)

Ikuti langkah ini jika anda ingin berkongsi keputusan perlawanan secara nyata/live dengan pengguna lain di GitHub Pages.

1. Layari Firebase Console dan cipta projek percuma baharu.

2. Cipta **Web App** dalam projek tersebut dan salin blok kod `firebaseConfig` anda.

3. Di menu sebelah kiri, pergi to **Firestore Database**, klik **Create Database** (pilih mod pengujian/test mode).

4. Buka kod fail HTML ini, cari baris **`GITHUB_FIREBASE_CONFIG`** (baris 520+) di bahagian bawah, gantikan dengan config anda, dan muat naik semula ke GitHub!
